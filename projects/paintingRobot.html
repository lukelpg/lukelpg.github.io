Painting Robot

  • Software Languages: Python, C++, ROS2, OpenCV
  • Hardware: Raspberry Pi, Arduino, Ender 3 Frame, Stepper Motors, Limit Switches, Custom 3D-Printed Brush and Paint Mounts, Original Cable Routing
  • Github URL: Project Link

Created Painting Robot using ROS2.

Converted an Ender 3 3D printer into a Painting Robot by replacing the stock motherboard with an Arduino connected to a Raspberry Pi running ROS2, enabling modular control and communication.

Programmed custom firmware on the Arduino to directly control the stepper motors and read limit switch feedback, while reusing the original Ender 3 cable routing for clean integration.

Designed and 3D-printed custom mounts for the paintbrush and paint containers, enabling reliable and repeatable mechanical operation of the painting process.

Developed ROS2 nodes for serial communication, motion control, and image processing with OpenCV to derive brush strokes from input images and translate them into robot trajectories.

Built a pipeline to transform digital images into layered painting instructions, demonstrating the integration of computer vision, robotics, motion planning, and custom mechanical design in a complete system.
